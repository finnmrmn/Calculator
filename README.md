
# Simple GUI Calculator

A self-study project for a GUI calculator that accepts text input.

# Calculator Language Features

Parentheses: ( )

Operators: +, -, unary +, unary -, *, /, mod, ^ (power)

Constants: e and pi

Trigonometry: sin, cos, tan, arcsin, arccos, arctan

Precision: double

Reports 'syntax error' when the parser fails.

# GUI
Text input.

Input-output history.

Reset button.

# Dependensies
GTK4

Alternatively, if GUI is unnecessary, copy the parser code into a test project and test it that way.
